---
title: Fahrenheit to Celsius
category: Number
---

**JavaScript version**

```js
const fahrenheitToCelsius = (fahrenheit) => ((fahrenheit - 32) * 5) / 9;
```

**TypeScript version**

```js
const fahrenheitToCelsius = (fahrenheit: Number) => ((fahrenheit - 32) * 5) / 9;
```

**Examples**

```js
console.log(fahrenheitToCelsius(77)); //25
```
