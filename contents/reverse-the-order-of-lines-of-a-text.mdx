title: Reverse the order of lines of a text
updated: '2021-10-13'
---

**JavaScript version**

```js
const reverseLines = (str) => str.split(/\r?\n/).reverse().join('\n');
```

**TypeScript version**

```js
const reverseLines = (str: string): string => str.split(/\r?\n/).reverse().join('\n');
```

**Examples**

```js
reverseLines(`one
two
three`);

/* Output */
/*
three
two
one
*/
```
